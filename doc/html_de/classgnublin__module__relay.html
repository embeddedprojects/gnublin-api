Ausführliche Beschreibung

Klasse für den zugriff auf das GNUBLIN module-relay.

Das GNUBLIN module-relay lässt sich mit Hilfe der gnublin_relay API ganz einfach ansteuern. Das Modul nutzt die I2C-Schnittstelle.

Beschreibung der Konstruktoren und Destruktoren

gnublin_module_relay::gnublin_module_relay ( )

Setze Standartwerte.

Setze standard i2c Adresse 0x20 und setze das ErrorFlag auf false.
