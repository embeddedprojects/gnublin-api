Ausführliche Beschreibung

Klasse für den Zugriff auf den SPI-Bus.

Diese Klasse ermöglicht das Senden und Empfangen von Daten über den SPI-Bus.

Beschreibung der Konstruktoren und Destruktoren

gnublin_spi::gnublin_spi ( )

Die Standart SPI-Treiber werden geladen, wenn sie noch nicht vorhanden sind.

Default chipselect: GNUBLIN: CS = 11 RASPBERRY PI: CS = 0
